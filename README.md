# BNUwork4

## 1. 实验目标
本实验旨在利用 Taichi 编程语言，从零实现一个基于光线投射的三维场景渲染器。主要目标包括：
* 理解并掌握局部光照的基本原理，实现环境光（Ambient）、漫反射（Diffuse）和镜面高光（Specular）的分量计算。
* 熟练掌握三维空间中的向量运算与隐式几何体（球体、圆锥）的光线求交算法。
* 掌握 Taichi 并行计算核函数（Kernel）的编写，并通过 UI 控件实现实时交互渲染。
* 进阶：实现 Blinn-Phong 光照模型升级，并增加基于暗影射线（Shadow Ray）的硬阴影投射。

---

## 2. 核心算法实现

### 2.1 隐式几何建模与深度测试
本实验未使用外部模型文件，而是通过数学方程的判别式计算射线与几何体的交点 $t$：
* **球体**：计算射线与球面的二次方程根。
* **圆锥**：结合了圆锥侧面的二次方程求交与底面的平面求交，并计算相应的法向量 $\mathbf{N}$。
* **深度竞争 (Z-buffer 逻辑)**：由于场景中存在多个物体，通过遍历所有几何体并选取最小的正数 $t$ 作为最终的击中点，确保了正确的视点遮挡关系。

### 2.2 光照模型：从 Phong 到 Blinn-Phong
最初实现了经典的 **Phong 光照模型**，通过计算理想反射向量 $\mathbf{R} = 2(\mathbf{N} \cdot \mathbf{L})\mathbf{N} - \mathbf{L}$ 来获取高光。
为解决极端角度下的视觉瑕疵，对模型进行了 **Blinn-Phong 升级**。引入了光源方向 $\mathbf{L}$ 与视线方向 $\mathbf{V}$ 的半程向量 $\mathbf{H}$：
$$\mathbf{H} = \frac{\mathbf{L} + \mathbf{V}}{|\mathbf{L} + \mathbf{V}|}$$
高光强度的计算由 $(\mathbf{R} \cdot \mathbf{V})^n$ 替换为 $(\mathbf{N} \cdot \mathbf{H})^n$。

### 2.3 硬阴影 (Hard Shadow)
在确定射线与物体的交点 $P$ 后，向光源方向发射一条“暗影射线”（Shadow Ray）。如果该射线在距离光源之前与场景中的其他几何体发生碰撞，则判定点 $P$ 位于阴影中。
* **自阴影消除 (Shadow Acne 避免)**：在发射暗影射线时，将起点沿着光线方向偏移了一个极小值 $\epsilon$（本代码中设为 $10^{-3}$），以避免由于浮点数精度导致的射线立刻与自身表面相交。处于阴影中的像素仅保留环境光（Ambient）分量。

## 3. 实验效果与视觉表现差异分析

### 3.1 材质参数的实时交互
通过 Taichi UI 模块提供了实时的滑块控制。
* 调节 $K_a$、$K_d$ 可以明显观察到环境光底色与漫反射对物体体积感的塑造。
* 调节 $K_s$ 和 Shininess 参数，可以模拟出不同粗糙度的材质（如哑光塑料到高光金属）。

### 3.2 重点分析：Phong vs Blinn-Phong 的边缘高光截断
当光源位于侧后方，光线以较大的入射角（掠射角）擦过物体表面时，两种光照模型表现出了显著的视觉差异。



* **Phong 模型的高光截断 (Cutoff Artifact)**：
  在极端角度下，理想反射向量 $\mathbf{R}$ 偏转到了视线的背面，导致点乘 $\mathbf{R} \cdot \mathbf{V}$ 变为负数并被强制截断为 0。视觉上（见对比图），**高光斑在物体边缘会突兀地消失，形成一条极其锐利、不自然的直线分界**。
* **Blinn-Phong 模型的平滑过渡**：
  由于采用的是法线 $\mathbf{N}$ 与半程向量 $\mathbf{H}$ 的夹角，无论视线和光线的相对位置如何，该夹角始终保持在合理范围内。因此，即使在最边缘处，**高光光斑也能顺着几何体的表面呈现出平滑、自然的向外衰减**，完美消除了截断瑕疵。

## 4. 实验总结
通过本次实验，成功使用纯代码（不依赖现有引擎）驱动构建了 3D 渲染管线的核心部分。不仅掌握了 Taichi 高性能向量计算的编写规范，更通过 Blinn-Phong 和阴影射线的拓展，深刻体会到了计算机图形学中“通过数学优化解决视觉 Bug（如高光截断）”和“通过光线追踪模拟物理真实（如阴影偏置避免 Acne）”的美妙过程。
